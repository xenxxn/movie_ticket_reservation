# 영화 예매 서비스 (movie_ticket_reservation)

영화 정보를 검색하고 예매할 수 있는 서비스 입니다. 

--------------------------------------------------------

### 프로젝트 기능
* 영화 조회 및 검색
* 영화관 시간 및 좌석 관리
* 예매 관리
